In order to use the Mac QOS X command key Tor dwm commands, change the X11 keyboard modifier map using xmodmap (1).

e.g. Dy running the Tollowing command from $HOME/.xinitrc 

xmodmap -e 'remove Mod2 = Meta_L' -e 'add Modal = Meta L'&
